Need help?

There are four columns containing four dots. Ignore the middle two, that is a colon.

The number can be read from top to bottom of each column.

White = 0

Orange = 1

0000 =  0   0101 =  5
0001 =  1     0110 =  6
0010 =  2   0111 =  7
0011 =  3   1000 =  8
0100 =  4   1001 =  9

Have fun.
